Regression models

To get an idea of the estimated effect of different predictors on subjects judgments, I ran some regression models (logistic regression, logit-link). All models have a random effect for each stimulus and another one for each subject.You can think of this as an estimated offset from the mean for each item and subject, to allow for variation that may be due to differences across items/subjects. Models considered:

  1. Intercept-only model
  2. Intercept + superglue_entailment (TRUE/FALSE)
  3. Intercept + superglue_entailment (TRUE/FALSE) + referential_anchoring (TRUE/FALSE) + lexical_pragmatics (TRUE/FALSE)
  4. Intercept + superglue_entailment (TRUE/FALSE) + lexical_pragmatics (TRUE/FALSE)
  5. Intercept + superglue_entailment (TRUE/FALSE) + referential_anchoring (TRUE/FALSE)
  6. Intercept + superglue_entailment (TRUE/FALSE) + either_referential_anchoring_or_lexical_pragmatics (TRUE/FALSE)
  7. Intercept + referential_anchoring (TRUE/FALSE) + lexical_pragmatics (TRUE/FALSE)

Model ranking

Besides estimating effects, I also wanted to see if there is one of these models that is clearly better/worse than the others. The main reason for doing so is to check if some predictor(s) are doing most of the predictive heavy lifting / whether some predictors are not very useful. Model ranking is in terms of leave-one-out performance (expected log pointwise predictive density, i.e., how well out-of-sample data would be accounted for by each model). The best model is at the top and ranking is in terms of difference to the best model (elpd_diff). Taking into consideration that the ELPD differences are small (even moreso if factoring in the standard error of this difference, se_diff), the main thing we learn is that most models are similar. Id say that only Model 1 and Model 7 are disprefered but those also have fewer or less informative predictors so this is not very surprising.

Comparison of aggregation methods

Another thing we wondered about is to which extent the SuperGlue methodology (taking the mode of MTurkers as the gold standard) is hiding variation in judgements. Here are two visualizations that show that this is indeed the case. In both cases, were plotting the absolute value of the difference between the mode (superglue methodology) and the mean judgment for each stimulus. Values close to 0 indicate that theres little to no difference in the aggregation method. Larger values are consequently of items where the mode and the mean differ more.

Visualization 1 plots the data using two boxplots. One for stimuli labeled as TRUE in superglue and the other for those labeled false. Visualization 2 further divides the data between cases where we labeled stimuli as being a case of either referential anchoring or lexical pragmatics vs.cases where neither label applied.

We see two main things:

  1. Plot1: There is a lot of variation. Theres a mean difference of about 0.12 between the mode and the mean, and a few items go far beyond that. In this regard, there is no difference between TRUE or FALSE superglue entailments
  2. Plot2: Items we had marked as either anchoring or pragmatics cases evince more variation than those that do not.
