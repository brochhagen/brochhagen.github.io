As a sanity-check, we can double-check with Bayesian hypothesis testing:

Code 
q1_formula <- 'controlsubject : grammaticalungrammatical : distractormismatch > 0'
q2_formula <- 'controlsubject : grammaticalungrammatical : distractormismatch = 0'
q3_formula <- 'controlsubject > 0'
q4_formula <- 'controlsubject = 0'
hyp <- hypothesis(m, c(q1 = q1_formula,
                       q2 = q2_formula,
                       q3 = q3_formula,
                       q4 = q4_formula)
                       )

print(hyp)
