df_exp <- readr::read_csv(paste0(relative_path, 'data/Experiment_3_eye_tracking/df_exp.csv')) %>%
          mutate(qtile = cut(presentation_order, breaks=quantile(presentation_order), include=T, labels=F))

This notebook zooms in at data from Experiment 3, region 6 to double-check whether subjects behaved differently across the experiment. To do so, I fit, for each measure and region, two models: one with only data from the QI of presentation; the other with only data from the QIV of presentation. All models have trial order as a fixed and random effect.

The summary is that only FP for QI changes. There, we find effects for CxGxD, G and D. The other measures do not show anything though so it may as well be noise.

df_region <- df_exp %>% 
            filter(region == 6) %>%
            filter(qtile == 1)
r_path <- paste0(relative_path,'/data/Experiment_3_eye_tracking/models/r6/qI-fp-')

mQI <- brm(data    = df_region,
          formula = log.first_pass ~ control * grammatical * distractor + presentation_order + (control * grammatical * distractor + presentation_order | participant) + (control *grammatical * distractor + presentation_order |item_identifier),
          chains  = 4,
          iter    = 2000,
          warmup  = 1000,
          file    = paste0(r_path, 'qi')
          )
