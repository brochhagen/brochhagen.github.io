center_vars <- function(df_in){

  df_in$c.memory_pcu <- df_in$memory_pcu - mean(df_in$memory_pcu)

  df_in$c.rating <- df_in$rating - mean(df_in$rating)

  df_in$c.NP_freq <- df_in$NP_freq - mean(df_in$NP_freq)

  df_in$c.freq_guipuzkoa <- df_in$freq_guipuzkoa - mean(df_in$freq_guipuzkoa)

  df_in$c.NP_mean_age <- df_in$NP_mean_age - mean(df_in$NP_mean_age)

  df_in$c.Nchar <- df_in$Nchar - mean(df_in$Nchar)

  df_in$c.verb_oind_freq <- df_in$verb_oind_freq - mean(df_in$verb_oind_freq)

  df_in$c.control_preferences <- df_in$control_preferences - mean(df_in$control_preferences)

  return(df_in)
}

In terms of ELPD, the model with control is better than the one that has only the intercept. OK. So this is a lot more interesting. We can see that the best models are m7 (fixed effect: grammatical * distractor * control) closely followed by m6 (grammatical distractor*) and then m3 (fixed effect: distractor). This is in line with what we discussed, for whatever reason, distractor seems to be best in predicting log.first_pass but having information about grammaticality also helps. The information about control is pretty secondary.

Summary (Go past: region 7)

There is a small but pretty robust effect on the interaction of distractor and grammaticality (and control). This is in line with there being something to the interaction between grammaticality and distractor, but apparently this is particularly clear for a model that also has control information (though the model without it is almost as good… it’s essentially a tie.)
